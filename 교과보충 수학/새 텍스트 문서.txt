벡터 => 시작점과 끝점이 있어 방향과 크기(힘)을 표현할 수 있는 놈
평면벡터 => 시점을 원점으로 고정한 벡터, 이 때 힘과 방향이 같은 모든 벡터들은 유일해진다
단위벡터 => 크기가 1인 벡터, 즉 방향만 알 수 있다

벡터의 길이는 절댓값 기호로 표현한다 (|A|)

벡터가 서로 같다 => 방향과 크기가 모두 같았을 때
벡터의 음수 => 벡터의 방향을 음수로 함, 즉 시점과 종점이 뒤바뀜

벡터의 덧셈 => 힘과 방향을 모두 더하는 것, a + b = a.힘 + b.힘 // a.방향 + b.방향
=> a 벡터의 종점부터 b벡터를 그린다.
=> 이 때 a 벡터의 시점부터 b 벡터의 종점 => a + b 벡터

벡터의 뺄셈 => a + (-b) == a 벡터 + b 벡터의 반대 방향
벡터의 곱셉 => 방향은 그대로 힘만 n 배수로 한다
벡터의 평행 => 벡터는 방향이 반대여도 평행이라 말할 수 있다, 즉 a와 -a 는 평행

Lerp => lerp(a, b, t)를 했을 때 구할 수 있는 점 P 의 진행방향 => b - a;
=> 즉 점 P = t * AB => t(b - a)
=> P 벡터는 a + t(b - a) => (1 - t)a + t * b

벡터의 내적 => 각 성분끼리 곱한 후 덧셈
=> a = (x, y), b = (p, q)
=> a 내적 b = x * p + y * q = xp + yq
